# Relay settings

Use this page to configure global relay behavior, monitoring rules, and pricing configuration.

## Page purpose

- Adjust relay-wide settings.
- Maintain monitoring configuration.
- Control queue behavior.
- Edit pricing configuration and model pricing rows.

## What you will see

### Settings groups

- Collapsible sections for different configuration domains.
- Basic relay settings.
- Monitor configuration table.
- Queue settings.

### Pricing area

- Pricing configuration editor.
- Editable model-pricing rows.
- Import, export, and copy actions.

## Common actions

1. Open the section that matches the configuration you need.
2. Adjust the relevant relay or monitor setting.
3. Save per section where required.
4. Import or export pricing data when doing larger configuration updates.
5. Review model pricing carefully before publishing cost changes.

## Notes

- Large pricing changes are easier to manage through import/export workflows.
- Monitoring configuration directly influences status and alert views.
- Use caution when changing queue or routing behavior in production.
