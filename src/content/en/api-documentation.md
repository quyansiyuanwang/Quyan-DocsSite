# API documentation

Use this page as the public reference hub for endpoint addresses, base URLs, and pricing-related displays.

## Page purpose

- Show the main API entry points clearly.
- Let users copy ready-to-use endpoint URLs.
- Display response-field guidance and pricing views.

## What you will see

### Endpoint tab

- Base URL.
- Provider URL.
- Balance URL.
- Platform-balance URL.
- Copy buttons.
- Option to show the full endpoint path.

### Pricing tab

- Pricing list with filters.
- Refresh action.
- Model or provider-related views.
- Toggle options such as custom multiplier or token-unit display.

## Common actions

1. Copy the correct base or endpoint URL for your integration.
2. Use the full-endpoint toggle if you need the exact complete path.
3. Check response-field hints before building a client.
4. Open the pricing tab when you need model cost comparison.

## Notes

- This page is reference-oriented and pairs well with Swagger.
- If you need interactive endpoint testing, use the dedicated Swagger site.
- For account-specific secrets or keys, return to access-key or relay-token pages.
