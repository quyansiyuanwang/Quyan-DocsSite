# 兑换码管理

此页面用于创建、复制和追踪兑换码。

## 页面用途

- 批量生成兑换码。
- 查看兑换码状态。
- 删除不再需要的兑换码。
- 支持用户充值或活动发放。

## 你会看到什么

### 列表信息

- 兑换码内容。
- 未使用、已使用、已过期等状态。
- 分页列表。

### 管理动作

- 批量创建。
- 点击复制。
- 删除兑换码。

## 常见操作

1. 创建前先设置金额、数量和过期时间。
2. 批量生成后复制给运营或发放对象。
3. 在用户反馈无法兑换时，先检查状态是否已使用或过期。
4. 清理错误生成或不再需要的兑换码。

## 说明

- 兑换码通常会影响余额或权益。
- 状态异常时，可联动查看余额记录与业务日志。
- 用户输入兑换码的页面通常在 `balance-history`。
