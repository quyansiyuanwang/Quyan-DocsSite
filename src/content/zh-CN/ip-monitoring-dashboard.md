# IP 监控面板

此页面用于查看 IP 风险数据、黑名单状态与白名单控制。

## 页面用途

- 查询当前 IP 状态。
- 查看错误权重和衰减表现。
- 维护黑名单与白名单记录。
- 识别高风险 IP 趋势。

## 你会看到什么

### 总览与查询工具

- 总览标签页。
- IP 状态查询。
- 封禁状态、错误权重、当前等级与阈值。
- 衰减估计和错误拆分。

### 黑名单管理

- 高风险或高封禁 IP 列表。
- 搜索、新增、编辑、解封动作。

### 白名单管理

- 新增和移除白名单项。
- 需要权限控制的敏感操作。

## 常见操作

1. 先查询可疑 IP。
2. 查看当前等级是否符合观察到的行为。
3. 需要执行限制时新增或编辑黑名单记录。
4. 仅对可信例外对象使用白名单。

## 说明

- 白名单应谨慎使用，因为它可能绕过部分防护策略。
- 衰减估计有助于判断封禁是否会随着时间自然缓解。
- 此页面与系统日志、服务端配置联动使用效果最好。
