# 系统统计

此页面用于监控运行健康度、资源占用和服务摘要信息。

## 页面用途

- 查看当前服务状态。
- 观察运行时长和时间指标。
- 检查内存、CPU 与基础设施摘要。
- 手动刷新或自动刷新指标。

## 你会看到什么

### 核心指标

- 服务时间与运行时长。
- 应用级计数。
- 内存使用情况。
- CPU 使用情况。

### 环境摘要

- 运行时信息。
- Redis 或基础设施状态。
- 配置摘要。

### 控制项

- 全部刷新按钮。
- 自动刷新时间间隔选择。
- 自动刷新状态提示。

## 常见操作

1. 对比数据前先刷新所有指标。
2. 在故障观察期间开启自动刷新。
3. 当性能异常时对比内存和 CPU 变化趋势。
4. 升级问题前先查看基础设施摘要。

## 说明

- 自动刷新适合事件处理中使用，日常查看未必需要长期开启。
- 运行时长正常并不代表所有子系统都完全健康。
- 深度排查时建议结合日志页和消耗统计页一起看。
