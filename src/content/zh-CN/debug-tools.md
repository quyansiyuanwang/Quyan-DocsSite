# 调试页

此页面用于诊断、测试和支持侧排障操作。

## 页面用途

- 查看前后端构建信息。
- 触发认证或转发相关测试动作。
- 快速打开相关文档入口。
- 执行环境特定的调试工具。

## 你会看到什么

### 信息区域

- 前端构建信息。
- 后端构建信息。
- 前往文档或 OpenAPI 参考的快捷入口。

### 测试与工具动作

- 认证测试动作。
- trusted-window 清理动作。
- 权限加载、展示与清理动作。
- 转发表单测试。
- 通用测试按钮。

### 季节性或预览工具

- 启用时可见的 April 预览模拟/打开/关闭控制。

## 常见操作

1. 对比环境前先查看构建信息。
2. 用最小必要的测试动作复现问题。
3. 调试会话问题时清理临时窗口或权限状态。
4. 需要更多说明时从此页跳转到文档或 API 参考。

## 说明

- 此页面主要面向高级用户、支持人员或开发人员。
- 在生产环境中避免执行无关测试动作。
- 调试结果通常还需要结合日志或监控页面确认。
