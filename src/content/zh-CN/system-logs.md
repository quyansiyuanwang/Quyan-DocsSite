# 系统日志

此页面用于检查 API 日志和服务端日志文件。

## 页面用途

- 筛选并查看 API 请求日志。
- 检查请求与响应详情。
- 浏览服务端日志文件并进行内容搜索。
- 支持后端与基础设施问题排查。

## 你会看到什么

### API 日志标签页

- 按方法、路径、状态、用户、请求 ID、IP、日期筛选。
- 可展开的日志行。
- 请求与响应载荷。
- 请求头、响应头和元数据。
- 分页控件。

### 服务端日志标签页

- 日志文件列表。
- 文件内容查看器。
- 文件内搜索。
- 行数限制控制。

## 常见操作

1. 先通过筛选锁定目标请求或时间窗口。
2. 展开日志行查看载荷和请求头细节。
3. 当问题超出单次请求范围时，转到服务端文件搜索。
4. 跨工具排查时使用请求 ID 进行关联。

## 说明

- 请求和响应载荷可能包含敏感运维数据。
- 展开单行适合链路分析，文件搜索更适合查找反复出现的问题模式。
- 这是诊断转发、认证、计费或监控问题的核心页面之一。
