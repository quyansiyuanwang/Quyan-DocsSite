# OJ 用量统计

此页面用于查看 OJ 提交器流量的请求量、Token 用量和费用明细。

## 页面用途

- 分析 OJ 提交器在一段时间内的使用情况。
- 比较请求、Token 和费用总量。
- 查看详细使用记录。

## 你会看到什么

### 筛选与汇总

- 时间范围筛选。
- 请求数、Token、费用、平均 Token 等汇总指标。

### 用量列表

- 表格或卡片列表。
- 模型、题目、Token、费用等字段。
- 分页控件。

### 详情视图

- 完整题目内容。
- 完整回答内容。

## 常见操作

1. 先设置要查看的时间范围。
2. 先观察顶部汇总，再进入明细。
3. 按模型或题目模式对比记录。
4. 需要完整上下文时打开详情记录。

## 说明

- 详情中可能包含敏感题目或回答内容。
- 费用突变更适合结合价格配置一起解释。
- 若要追踪是谁在使用，建议回到 API Key 管理页联动查看。
