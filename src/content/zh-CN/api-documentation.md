# API 文档

此页面是公开接口地址、基础 URL 和价格展示的参考入口。

## 页面用途

- 清晰展示主要 API 入口。
- 让用户直接复制可用的接口 URL。
- 展示响应字段说明与价格视图。

## 你会看到什么

### 端点标签页

- Base URL。
- Provider URL。
- Balance URL。
- Platform-balance URL。
- 复制按钮。
- 显示完整端点路径的切换项。

### 价格标签页

- 带筛选的价格列表。
- 刷新按钮。
- 模型或供应商相关视图。
- 自定义倍率、token 单位等切换项。

## 常见操作

1. 复制集成所需的基础地址或具体端点地址。
2. 需要完整路径时，开启完整端点显示。
3. 编写客户端前先查看响应字段说明。
4. 需要比较模型成本时进入价格标签页。

## 说明

- 此页面偏向参考说明，适合和 Swagger 配合使用。
- 如果需要交互式调试接口，请使用专门的 Swagger 站点。
- 如需账号级密钥或转发 Token，请返回对应管理页面。
