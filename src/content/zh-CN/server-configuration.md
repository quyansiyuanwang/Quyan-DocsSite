# 服务端配置

此页面用于维护全局服务端设置。

## 页面用途

- 配置注册行为。
- 维护计费设置。
- 管理 SMTP 与邮件发送配置。
- 调整错误衰减与 IP 封禁阈值。

## 你会看到什么

### 配置区块

- 注册设置。
- 计费设置。
- SMTP 设置。
- 错误衰减设置。
- IP 封禁阈值与封禁时长。

### 编辑工具

- 按区块折叠展示。
- 可编辑权重表。
- 每个区块独立保存按钮。

## 常见操作

1. 展开需要调整的区块。
2. 只修改当前任务相关的参数。
3. 按区块逐步保存。
4. 重大规则变更后再检查相关监控表现。

## 说明

- 此处的小改动可能影响注册、计费、邮件发送和防滥用策略。
- SMTP 配置错误往往会在通知或找回密码流程中暴露出来。
- IP 封禁和衰减调整应结合监控数据进行。
