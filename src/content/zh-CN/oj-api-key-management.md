# OJ API Key 管理

此页面用于维护 OJ 提交器工作流所需的 API Key。

## 页面用途

- 创建和管理 OJ 提交器 API Key。
- 查看绑定渠道与用量。
- 安全复制或轮换密钥。

## 你会看到什么

### Key 列表

- Key 名称。
- 截断或完整密钥显示。
- 绑定渠道。
- 请求次数与累计 Token。
- 最后使用时间与到期时间。

### 管理动作

- 新建 Key。
- 编辑 Key。
- 复制 Key。
- 删除 Key。

## 常见操作

1. 为提交器或具体场景创建易识别名称的 Key。
2. 创建后立即复制并妥善保存。
3. 排查请求失败前先检查绑定渠道和到期时间。
4. 不再使用时编辑或删除对应 Key。

## 说明

- 新生成的密钥明文可能只会展示一次。
- OJ API Key 与普通账号访问密钥是分开的。
- 用量异常时，配合用量统计页更容易定位问题。
